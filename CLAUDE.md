# あなたの役割

あなたはプロのWebデザイナー兼フロントエンドエンジニアです。

## プロジェクト概要

上の山彩果園（福島県郡山市の梨農園）のランディングページ。フレームワーク・ビルドツール不使用の静的サイト。

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | ページ全体のHTML（セクション構成・コンテンツ） |
| `style.css` | 全スタイル定義（クラスベース） |
| `main.js` | スムーススクロール・ハンバーガーメニュー |
| `assets/` | 画像素材（`s1.png`〜`s82.png` はアイコン類、`photo_*.png` は写真） |

## 開発・確認方法

ビルド不要。ブラウザで `index.html` を直接開くか、ローカルサーバーを起動して確認する。

```
# Python がある場合
python -m http.server 8080

# Node.js がある場合
npx serve .
```

## アーキテクチャの注意点

**スタイルの使い分け**
- レイアウト・インタラクション（hover, responsive）は `style.css` のクラスで管理
- セクション固有の色・余白・一点もののスタイルは `index.html` 内のインライン `style=""` で直接指定している（混在は意図的な設計）

**カラーパレット（変更時の一貫性のため）**
- メイングリーン: `#5e8c3a` / `#4d6b2f` / `#7bb04f`
- アクセント（ゴールド）: `#c9b15f` / `#cdb96a`
- テキスト: `#514c3e` / `#5c5849`

**フォント**
- 見出し: `Zen Maru Gothic`（セクションタイトル・カード見出し）
- キャッチコピー: `Shippori Mincho`（ヒーロー `h1` のみ）
- 本文: `Zen Kaku Gothic New`

**レスポンシブ対応ブレークポイント**
- `1024px`: タブレット（グリッド縮小、ナビ非表示→ハンバーガー）
- `768px`: モバイル（各グリッドを1カラム化）
- `480px`: 小型スマートフォン（フォントサイズ縮小）
